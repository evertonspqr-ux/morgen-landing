# Mídia do mockup — preencha aqui

Troque os arquivos mantendo o **mesmo nome**, ou atualize os `src` no `index.html`.

| Slot no site | Arquivo sugerido | Tipo | Notas |
|--------------|------------------|------|--------|
| Hero 3D / produto | `hero-3d.png` ou `.webm` | PNG/WebM | 1600×1200 ou 16:9, fundo navy ou transparente |
| Hero ambient foto | `hero-ambient.jpg` | JPG | Escritório / operação real, escura |
| Vídeo case | `case-loop.mp4` + `case-poster.jpg` | MP4 + poster | Loop mudo 8–15s |
| Foto problema | `pain-ops.jpg` | JPG | Operação caótica / atendimento |
| Foto solução | `solucao-dash.jpg` | JPG | Dashboard / fluxo organizado |
| 3D bento 1–3 | `bento-1.png` … `bento-3.png` | PNG | Ícones/objetos 3D, fundo transparente |
| Método visual | `metodo-diagram.png` | PNG | Diagrama M.O.R.G.E.N. |
| Depoimento avatars | `avatar-1.jpg` … | JPG | 400×400 |
| Logos clientes | `logo-client-1.svg` … | SVG | Monocromo claro |
| CTA fundo | `cta-bg.jpg` | JPG | Wide 2400×1000 |

Enquanto não houver arquivo, o site mostra **slots tracejados** com o nome do slot.
