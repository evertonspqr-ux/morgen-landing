# Morgen Landing

Landing / constituição visual da **Morgen** (export Open Design → HTML).

## Preview local

```bash
# abrir no browser
xdg-open index.html
# ou
python -m http.server 8760
```

## Arquivos

- `index.html` — entry (manifest)
- `morgen-landing.html` — landing completa
- `logo-morgen.svg` — wordmark
- `brand-spec.md` — brand
- `DESIGN-HANDOFF.md` / `DESIGN-MANIFEST.json` — handoff
